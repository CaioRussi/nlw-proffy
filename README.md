## Instalação e uso

### API

```bash
$ git clone git@github.com:CaioRussi/nlw-proffy.git
$ cd nlw-proffy/api
$ npm install

# create sqlite database
$ npm run migrate:up

$ npm start
```

<br>

### Web

```bash
$ cd nlw-proffy/web
$ npm install
$ npm start
```

<br>

### Mobile

```bash
$ cd nlw-proffy/mobile
$ npm install
$ npm start

# Scan QRcode
```
